# Agile Principles & Frameworks

## What is Agile?

Agile is a way of managing projects and developing software that focuses on flexibility, collaboration, and continuous improvement. Unlike traditional project management, which usually follows a fixed plan from start to finish, Agile allows teams to adapt quickly when requirements or priorities change.

Traditional methods often complete work in separate phases, while Agile encourages regular feedback, smaller task deliveries, and close teamwork throughout the project.

---

## Core Values and Principles of Agile

The Agile Manifesto is built around four main values:

- Individuals and interactions over processes and tools
- Working software over comprehensive documentation
- Customer collaboration over contract negotiation
- Responding to change over following a fixed plan

Agile also focuses on teamwork, regular feedback, continuous improvement, and delivering value to users frequently.

---

## Scrum vs Kanban

Scrum and Kanban are both Agile frameworks, but they work differently.

Scrum uses fixed sprint cycles where tasks are planned and completed within a set period. It also includes meetings like sprint planning, standups, and retrospectives.

Kanban focuses more on continuous workflow. Tasks move through stages such as To Do, In Progress, and Done using a Kanban board. It is more flexible and helps teams manage changing priorities more easily.

Scrum is useful for structured sprint-based development, while Kanban works well for teams handling continuous tasks and fast-changing work.

---

## Why Focus Bear Uses Kanban

Focus Bear leans more towards Kanban because it provides flexibility and allows work to move continuously without waiting for sprint deadlines. It also helps the team manage priorities and track progress more efficiently.

## Reflection

One of the biggest benefits of Agile is that teams can adapt quickly to changes and improve continuously through feedback. Agile also encourages better communication and teamwork, which helps projects move more smoothly.
