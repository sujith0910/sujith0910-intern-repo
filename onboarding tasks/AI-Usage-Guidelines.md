# AI Usage Guidelines
## 1. AI tools used in my role
- Debugging errors and understanding issues.
- Writing README files and documentation
- Getting ideas for UI/UX improvements
- Learning new concepts quickly

---

## 2. Benefits and risks of AI
### Benefits
- Saves time, especially for repetitive tasks. Helps when stuck on a problem  
- Gives quick explanations and examples. Improves overall productivity  

### Risks
- Sometimes it gives wrong or outdated answers and can make you depend too much on AI.  
- Risk of sharing sensitive data by mistake  
- Code may not always follow best practices  

---

## 3. What should never be shared with AI
- Company or confidential data and any sensitive business information
- API keys, passwords, credentials 
- Private repositories or internal code  

---

## 4. How I will validate AI output
- Check official documentation and compare with other reliable sources before finalising.  
- Need to read and understand the logic instead of blindly using it  

---

## 5. When to use AI and my own skills
### I will use AI when:
- I’m stuck and need direction  
- I need examples or quick help  
- I want to speed up repetitive tasks  

### I will use my own skills when:
- Working on core logic  
- Making design decisions  
- Writing final/production code

---

## 6. How I avoid over-relying on AI
- Try solving problems first on my own  
- Use AI only as support, not as main solution  
- Always review and edit AI output and practice coding without AI regularly  

---

## 7. How I will ensure data privacy
- Never copy-paste sensitive or confidential info. Always double-check what I input into AI tools  
- Follow company guidelines.Must use safe, non-sensitive data  

---

## 8&9. Practical task
### Task:
- Used AI to improve a Git commit message and documentation wording  

### What I observed:
- AI helped make the message clearer and more professional.It suggested better wording and structure. Some parts felt too generic, so I had to improve them. I edited it to match the actual work I did  

### Conclusion:
- AI is useful for improving clarity and writing.But it still needs human input to make it accurate and relevant  

---

## 10. Best practice
- Use AI to improve clarity, but always customise it to match my actual work
